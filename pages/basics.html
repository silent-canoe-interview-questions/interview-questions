SQL Basics

Introduction to Databases

A database is an organized collection of structured information or data, typically stored electronically in a computer system. SQL Server is Microsoft's relational database management system (RDBMS) that helps you manage and analyze data efficiently.

Key Database Concepts

  • Database: A collection of related tables and other objects
  • Table: A collection of related data organized in rows and columns
  • Column: A vertical entity in a table that contains all information associated with a specific field
  • Row: A horizontal entity in a table that contains all information associated with a specific record
  • Primary Key: A column or combination of columns that uniquely identifies each row
  • Foreign Key: A column that creates a relationship between two tables

Creating Your First Database

Let's create a simple database using T-SQL (Transact-SQL) commands:

-- Create a new database
CREATE DATABASE LearningDB;

-- Switch to the new database
USE LearningDB;

-- Create a simple table
CREATE TABLE Employees (
    EmployeeID INT PRIMARY KEY IDENTITY(1,1),
    FirstName NVARCHAR(50),
    LastName NVARCHAR(50),
    Email NVARCHAR(100),
    HireDate DATE
);

Data Types in SQL Server

Data Type Description Example
INT Integer numbers 42, -17, 0
DECIMAL/NUMERIC Exact decimal numbers 123.45, -789.12
VARCHAR Variable-length string (ASCII) 'Hello'
NVARCHAR Variable-length string (Unicode) N'Hello'
DATE Date without time '2025-09-27'
DATETIME Date and time '2025-09-27 14:30:00'

Basic Query Structure

Here's a simple example of basic SQL query structure:

-- Insert sample data
INSERT INTO Employees (FirstName, LastName, Email, HireDate)
VALUES ('John', 'Doe', 'john.doe@example.com', '2025-01-15');

-- Basic SELECT query
SELECT FirstName, LastName
FROM Employees
WHERE HireDate >= '2025-01-01'
ORDER BY LastName;

Query Components

  • SELECT: Specifies which columns to retrieve
  • FROM: Indicates the source table(s)
  • WHERE: Filters the results based on conditions
  • ORDER BY: Sorts the results
